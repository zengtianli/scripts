================================================================================
                    旱地作物模块更新方案
================================================================================

【背景】

    之前：旱地作物数据是我们自己估算的，作物种类和需水量都是猜的
    现在：业主给了真实的灌溉定额数据，以业主数据为准


【老逻辑】（要废弃）

    数据文件：static_crops.txt
    
    格式：
        作物    75% mm/d    90% mm/d
        小麦    0.248       0.320
        玉米    0.375       0.482
        ...
    
    特点：
        - 全年每天都用同一个固定值
        - 作物列表是我们估的（13种）
        - 单位是 mm/d（日需水强度）
    
    计算：
        日需水量 = 日需水强度(mm/d) × 面积(km²) × 0.1


【新逻辑】（业主数据）

    数据文件：static_irrigation_quota.txt
    
    格式：
        作物    保证率  1月  2月  3月  ...  12月  合计
        蔬菜    0.5     0    0    0   ...  0     5
        蔬菜    0.75    0    0    0   ...  0     32
        蔬菜    0.9     0    0    0   ...  0     42
        小麦    0.5     0    0    1   ...  0     5
        ...
    
    特点：
        - 每个月的定额不同（更精确）
        - 作物列表以业主为准（目前8种）
        - 单位是 m³/亩（月灌溉定额）
        - 支持多个保证率（50%/75%/90%）
    
    计算：
        日需水量 = 当月定额(m³/亩) ÷ 当月天数 × 面积(km²) × 1500亩

【需要同步更新的文件】

    1. in_dry_crop_area.txt（旱地作物面积输入文件）
       
       老格式表头：
           平原名称  灌溉保证率  小麦  玉米  大豆  油菜  花生  ...
       
       新格式表头（按业主作物）：
           平原名称  灌溉保证率  小麦  油菜  蔬菜  瓜果  豆类
       
       说明：
           - 删除业主没给数据的作物（玉米、花生、马铃薯等）
           - 作物名称必须和 static_irrigation_quota.txt 完全一致
           - 面积数据需要业主提供或重新调查


【扩展性设计】

    业主后续增加作物，只需要：
    
    1. 在 static_irrigation_quota.txt 添加新作物的月度定额
       例如添加"棉花"：
           棉花    0.5     0    0    0   ...  0     xx
           棉花    0.75    0    0    0   ...  0     xx
           棉花    0.9     0    0    0   ...  0     xx
    
    2. 在 in_dry_crop_area.txt 添加对应列
       表头加上"棉花"，各平原填写面积
    
    代码不需要改动，自动支持新作物


【更新步骤】

    第一步：更新 in_dry_crop_area.txt
            - 删除旧作物列
            - 添加业主作物列
            - 填写正确的面积数据
    
    第二步：删除或备份旧文件
            - static_crops.txt → static_crops.txt.bak
    
    第三步：测试运行
            - python main.py
            - 检查 OUT_GGXS_C.txt 输出是否正确

【测试验证】

    丰惠平原区（蔬菜 1.0 km²，瓜果 0.5 km²）：
    - 蔬菜：9 m³/亩/月 ÷ 31天 × 1.0 km² × 1500亩 = 435.48 m³/d
    - 瓜果：36 m³/亩/月 ÷ 31天 × 0.5 km² × 1500亩 = 870.97 m³/d
    - 合计：1306.45 m³/d ✓ （与输出一致）
