# 纳污能力计算 - 公式与业务逻辑

## 一、河道纳污能力公式

$$W = 31.536 \times b \times (C_s - C_0 \times e^{-\frac{KL}{u}}) \times \frac{Q \times K \times L / u}{1 - e^{-\frac{KL}{u}}}$$

### 参数说明

| 符号 | 名称 | 单位 | 说明 |
|------|------|------|------|
| W | 纳污能力 | t/a | 河段每年可接纳的污染物量 |
| Cs | 目标浓度 | mg/L | 水质标准限值 |
| C0 | 初始浓度 | mg/L | 河段入口污染物浓度 |
| Q | 流量 | m³/s | 河道流量 |
| u | 流速 | m/s | 水流速度 |
| K | 衰减系数 | 1/s | 污染物自净速率 |
| L | 河段长度 | m | 计算河段长度 |
| b | 不均匀系数 | - | 污染物混合不均匀修正 |

### 流速计算公式

$$u = a \times Q^{\beta}$$

其中：
- a：流速系数
- β：流速指数

### 出流浓度公式

$$C_{out} = C_0 \times e^{-\frac{KL}{u}}$$

河段出口浓度等于入口浓度乘以衰减因子。


## 二、支流汇入的影响

### 流量混合

$$Q_{混合} = Q_{干流} + Q_{支流}$$

### 浓度混合（流量加权平均）

$$C_{混合} = \frac{Q_{干流} \times C_{干流} + Q_{支流} \times C_{支流}}{Q_{干流} + Q_{支流}}$$

## 四、正确计算流程

### 步骤1：计算干流段1（起点 → 支流汇入点）

输入：Q干流, L段1, C0干流
输出：W1, C1_out

### 步骤2：计算支流

输入：Q支流, L支流, C0支流
输出：W支流, C支流_out

### 步骤3：汇入点混合

```
    干流段1出流 ──→ ●
                    │  混合
    支流出流 ─────→ ●
                    │
                    ↓
               混合后流向段2
```

计算：
- Q_mix = Q干流 + Q支流
- C_mix = (Q干流 × C1_out + Q支流 × C支流_out) / Q_mix

### 步骤4：计算干流段2（汇入点 → 终点）

输入：Q_mix, L段2, C_mix
输出：W2, C2_out

### 步骤5：汇总

总纳污能力 = W1 + W支流 + W2
