# A2：流量变化（双正）

## 场景特征

| 特征 | 值 |
|------|-----|
| 段1 | **正** |
| 支流 | **正** |
| 段2 | **正** |
| 总体 | **正** |
| 出口 | **达标** |


## 场景参数对比

| 参数 | 基准A1 | 支流小 | 支流大 |
|------|--------|--------|--------|
| C0干 (mg/L) | 0.1 | 0.1 | 0.1 |
| C0支 (mg/L) | 0.1 | 0.1 | 0.1 |
| Q干 (m³/s) | 10 | 10 | 10 |
| Q支 (m³/s) | **2** | **0.5** | **8** |


## 计算结果对比

### 支流流量小（Q支=0.5）

| 河段 | 纳污能力 (t/a) | vs基准 |
|------|---------------|--------|
| 干流段1 | 189.7 | 0% |
| 支流 | 16.2 | -60% |
| 干流段2 | 206.7 | -10% |
| **总计** | **412.6** | **-10%** |

### 支流流量大（Q支=8）

| 河段 | 纳污能力 (t/a) | vs基准 |
|------|---------------|--------|
| 干流段1 | 189.7 | 0% |
| 支流 | 118.6 | +194% |
| 干流段2 | 315.0 | +38% |
| **总计** | **623.3** | **+36%** |


## 关键发现

### 1. 流量是主导因素

| 支流Q | 总纳污能力 | 变化 |
|-------|-----------|------|
| 0.5 | 412.6 | -10% |
| 2 | 459.2 | 基准 |
| 8 | 623.3 | +36% |

### 2. 流量影响机制

- **支流本身**：Q支增加 → W支线性增加
- **段2**：Q_mix增加 → 流速增加 → 自净能力增强 → W2增加
- **双重叠加效应**


## 浓度变化对比

```
支流小（Q=0.5）：
0.1 → 0.035 → 0.034 → 0.012
              ↑支流几乎无影响

支流大（Q=8）：
0.1 → 0.035 → 0.044 → 0.020
              ↑支流浓度略高，但仍达标
```


## 业务启示

1. **来水量管理**：枯水期纳污能力显著下降
2. **支流贡献**：大支流是提升纳污能力的关键
3. **规划建议**：优先保护大流量支流的水质
