河道功能区纳污能力计算 - 操作说明
=====================================

按钮清单：
- 初始化输入表
- 生成功能区列
- 生成干支流名
- 生成基础信息
- 生成逐日流量
- 开始计算
- 清除输入

=====================================

一、初始化输入表（自动生成）
----------------------------
点击「初始化输入表」按钮

自动生成内容：
- 创建「河道功能区-输入」Sheet
- A列说明：功能区数量→、方案个数→
- D列标签：功能区编号、功能区名、Cs、K(1/s)、b、a、β、干流总长L(m)、干流C0、支流数量、干流名

需要手动填写：
- B1：功能区数量（如：10）
- B2：方案个数（如：2）


二、生成功能区列（自动生成）
----------------------------
点击「生成功能区列」按钮

自动生成内容：
- 第1行E列起：功能区1、功能区2、功能区3 ...
- 高亮需填写的单元格（浅黄色）


三、填写功能区参数（手动填写）
------------------------------
E列起，逐列填写每个功能区的参数：

| 行号 | 内容 | 说明 |
|------|------|------|
| 第2行 | 功能区名 | 如：钱塘156、钱塘157 |
| 第3行 | Cs | 目标浓度，单位 mg/L |
| 第4行 | K | 衰减系数，单位 1/s |
| 第5行 | b | 不均匀系数 |
| 第6行 | a | 流速系数 |
| 第7行 | β | 流速指数 |
| 第8行 | 干流总长L | 干流长度，单位 m |
| 第9行 | 干流C0 | 干流入口浓度，单位 mg/L |
| 第10行 | 支流数量 | 0=无支流，1=1条支流，2=2条支流... |


四、生成干支流名（自动生成）
----------------------------
点击「生成干支流名」按钮

自动生成内容：
- 第11行：干流名（功能区名-0）
- 第12行起：支流信息（每条支流5行）
  - 分隔行：─支流N─
  - 支流N名：功能区名-N
  - 支流N长度L(m)：待填写
  - 支流N汇入位置(m)：待填写
  - 支流N浓度C0：待填写

需要手动填写（如有支流）：
- 支流长度L(m)
- 支流汇入位置(m)：在干流的第几米汇入
- 支流浓度C0


五、生成基础信息（自动生成）
----------------------------
点击「生成基础信息」按钮

自动创建「河道功能区-基础信息」Sheet，汇总所有干支流信息：

| 列 | 内容 |
|----|------|
| A列 | 功能区 |
| B列 | 名称（干流/支流名） |
| C列 | 类型（干流/支流） |
| D列 | 长度L(m) |
| E列 | 汇入位置(m) |
| F列 | Cs |
| G列 | C0 |
| H列 | K(1/s) |
| I列 | b |
| J列 | a |
| K列 | β |

可在此表中核对或微调参数。


六、生成逐日流量（自动生成 + 手动填写）
---------------------------------------
点击「生成逐日流量」按钮

根据方案数自动创建：
- 逐日流量（方案1）
- 逐日流量（方案2）
- ...

自动生成表头：
- A列：日期
- B列起：各干支流名称

需要手动填写：
- A列：日期（如 2020-01-01、2020-01-02 ...）
- B列起：各干支流逐日流量数据，单位 m³/s


七、开始计算（自动计算）
------------------------
点击「开始计算」按钮

前提条件：
1. 输入表参数已完整填写
2. 逐日流量表数据已填写

自动生成结果（每个方案两个Sheet）：

| Sheet名 | 内容 | 用途 |
|---------|------|------|
| 纳污能力过程（方案N） | 干流各段+支流+混合点+汇总 | 核验计算过程 |
| 纳污能力结果（方案N） | 只有干流各段+汇总 | 提交报告 |

结果列说明：
- 功能区
- 河段名称
- 类型（干流段/支流/混合/汇总）
- 长度L(m)
- 流量Q(m3/s)
- 入口C0
- 出口C
- 纳污能力(t/a)
- 备注（仅过程版有）


八、清除输入
------------
点击「清除输入」按钮

清除所有输入数据，回到初始状态。



=====================================
计算公式
=====================================
纳污能力：W = 31.536 × b × (Cs - C0 × exp(-KL/u)) × (Q×K×L/u) / (1 - exp(-KL/u))
流速：u = a × Q^β
出流浓度：C_out = C0 × exp(-K×L/u)
流量混合：Q_mix = Q1 + Q2
浓度混合：C_mix = (Q1×C1 + Q2×C2) / (Q1 + Q2)

汇总规则：只汇总干流段的纳污能力，支流仅展示不计入汇总。
