## 1. 需求背景

### 1.1 业务场景

水库发电调度是水资源优化配置的重要内容。湖南镇-黄坛口梯级水库调度模型已完成核心算法开发，需要提供可视化操作界面，让用户无需编程即可：
- 配置水库参数
- 导入来水/需水数据
- 运行调度计算
- 查看/导出计算结果

### 1.2 现状问题

目前模型使用 Python 脚本运行，存在以下问题：
- 需要修改代码才能调整参数
- 输入文件格式要求严格，用户易出错
- 无可视化界面，难以理解计算过程
- 结果为 CSV 文件，需额外处理才能分析

### 1.3 期望成果

开发一个 **桌面应用界面**，实现：
- 可视化参数配置
- 拖拽式数据导入
- 一键运行计算
- 结果可视化展示

## 8. 技术要求

### 8.1 技术选型建议

| 方案 | 技术栈 | 优点 | 缺点 |
|------|--------|------|------|
| **方案A（推荐）** | Python + PyQt/PySide | 复用现有模型代码，开发快 | 打包体积大 |
| 方案B | Python + Streamlit | 开发最快，可部署为Web | 不适合复杂交互 |
| 方案C | Electron + Python后端 | 界面精美，跨平台 | 开发周期长 |

### 8.2 核心模块复用

```python
# 直接复用现有计算核心
from HydroElectricityUpAddDown1029 import HydroElectricity, StructReservoir

# GUI层只负责：
# 1. 参数采集 → dict
# 2. 调用计算 → HydroElectricity.calculate()
# 3. 结果展示 → DataFrame / Chart
```

### 8.3 打包要求

- **Windows**：`.exe` 单文件或安装包
- **macOS**：`.app` 应用程序
