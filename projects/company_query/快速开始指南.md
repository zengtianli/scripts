# ⚡ 快速开始指南

## 🎯 5分钟上手

### 第1步：安装依赖（1分钟）

打开终端，进入项目目录，运行：

```bash
pip install pandas requests openpyxl
```

或者：
```bash
pip install -r requirements.txt
```

---

### 第2步：获取天眼查API密钥（需要注册）

#### 🔸 方式1：天眼查官网（推荐）

1. 访问：**https://open.tianyancha.com/**
2. 注册账号 → 实名认证
3. 控制台 → 应用管理 → 创建应用
4. 复制 **API Token**（类似：`abc123def456`）

#### 🔸 方式2：阿里云市场（备选）

1. 访问：**https://market.aliyun.com/**
2. 搜索"企业信息查询"或"天眼查"
3. 购买API服务（通常有免费试用）

#### 🔸 方式3：暂时跳过（仅用智能识别）

如果暂时没有API密钥：
- 可以先跳过，工具仍可识别政府机关、学校、医院
- 但无法查询企业工商信息

---

### 第3步：配置API密钥（1分钟）

打开 **`config.py`** 文件，找到第8行：

```python
TIANYANCHA_API_TOKEN = "your_api_token_here"
```

替换为您的API Token：

```python
TIANYANCHA_API_TOKEN = "abc123def456ghi789"  # 替换为您的实际密钥
```

保存文件。

---

### 第4步：测试运行（1分钟）

先运行测试脚本，确保配置正确：

```bash
python test_query.py
```

如果看到 `✅ 测试通过！` 说明一切正常。

---

### 第5步：正式查询（1分钟）

运行主程序：

```bash
python company_industry_query.py
```

等待查询完成，查看生成的 **`企业信息查询结果.xlsx`** 文件。

## ❓ 常见问题速查

### Q: 提示"未配置API Token"？
**A:** 打开 `config.py`，修改第8行的API Token。

### Q: 出现"API返回错误"？
**A:** 检查：
1. API Token是否正确
2. API额度是否用完
3. 网络是否正常

### Q: 某些企业查不到？
**A:** 正常现象：
- 政府机关、事业单位没有工商信息（会自动识别）
- 企业名称错误或已注销

### Q: 如何查看API剩余次数？
**A:** 登录天眼查平台 → 控制台 → 用量统计

### Q: API费用多少？
**A:** 
- 企业基本信息：约 0.5-1元/次
- 通常有 100-500次 免费额度
