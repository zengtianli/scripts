# 纳污能力业务

> 河道纳污能力计算与场景分析

## 什么是纳污能力？

**纳污能力**是指在满足水功能区水质目标的前提下，河段能够接纳的污染物最大量（单位：t/a）。

$$W = 31.536 \times b \times (C_s - C_0 \times e^{-\frac{KL}{u}}) \times \frac{Q \times K \times L / u}{1 - e^{-\frac{KL}{u}}}$$


## 场景分类

本文档按照 **段1（干流上段）** 和 **支流** 的纳污能力正负，将场景分为4大类：

| 大类 | 段1 | 支流 | 说明 |
|------|-----|------|------|
| **A** | + | + | 双正（正常情况） |
| **B** | - | + | 干负支正（干流超标） |
| **C** | + | - | 干正支负（支流超标） |
| **D** | - | - | 双负（都超标） |

每类下按严重程度细分为 3 个子场景：

| 严重程度 | 段2 | 总体 | 出口 |
|----------|-----|------|------|
| 轻度 | + | + | 达标 |
| 中度 | + | - | 达标 |
| 严重 | - | - | 超标 |

| 级别 | 条件 | 措施 |
|------|------|------|
| 🟢 绿色 | 总体 > 100 t/a | 正常管理 |
