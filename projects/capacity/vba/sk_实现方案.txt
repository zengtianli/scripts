水库纳污能力 VBA 实现方案
=========================

一、文件命名
-----------
sk_公共.bas        ← 公共函数
sk_输入准备.bas    ← 初始化输入表、生成功能区列
sk_数据生成.bas    ← 生成基础信息、生成逐日库容
sk_计算.bas        ← 计算水库纳污能力

二、核心公式
-----------
W = 31.536 × K × V × Cs × b

其中：
- W：纳污能力 (t/a)
- K：污染物综合衰减系数 (1/s)
- V：库容 (m³)
- Cs：目标浓度 (mg/L)
- b：不均匀系数

三、与河道的区别
---------------
| 项目 | 河道(hd_) | 水库(sk_) |
|------|-----------|-----------|
| 核心变量 | 流量Q、流速u | 库容V |
| 公式 | 含衰减指数exp() | 简单乘法 |
| 分段 | 需要（支流汇入） | 不需要 |
| 时间 | 自然年(1-12月) | 水文年(4月→次年3月) |
| 输入数据 | 逐日流量 | 逐日库容 |

四、输入表布局（水库功能区-输入）
--------------------------------
| 行号 | A列 | B列 | D列 | E列起 |
|------|-----|-----|-----|-------|
| 1 | 功能区数量→ | (数量) | 功能区编号→ | 功能区1,2,3... |
| 2 | 方案个数→ | (数量) | 功能区名 | (名称) |
| 3 | | | Cs | (目标浓度) |
| 4 | | | K(1/s) | (衰减系数) |
| 5 | | | b | (不均匀系数) |

注：比河道简单，没有干流长度、C0、支流等

五、基础信息表（水库功能区-基础信息）
------------------------------------
| 功能区 | 名称 | Cs | K(1/s) | b |
|--------|------|-----|--------|---|

六、逐日库容表（水库逐日库容-方案N）
-----------------------------------
| 日期 | 功能区1 | 功能区2 | ... |
|------|---------|---------|-----|
| 2020-04-01 | 45564115 | 40456955 | ... |

七、输出结果（水库纳污能力-方案N）
---------------------------------
按水文年顺序（4月→3月）：

| 功能区 | 名称 | 4月 | 5月 | ... | 2月 | 3月 | 年合计(t/a) |
|--------|------|-----|-----|-----|-----|-----|-------------|

八、按钮清单
-----------
1. 初始化输入表 ← sk_输入准备.bas
2. 生成功能区列 ← sk_输入准备.bas
3. 生成基础信息 ← sk_数据生成.bas
4. 生成逐日库容 ← sk_数据生成.bas
5. 开始计算     ← sk_计算.bas
6. 清除输入     ← sk_输入准备.bas

九、计算逻辑
-----------
1. 读取基础信息（名称、Cs、K、b）
2. 读取逐日库容
3. 按水文年分组：
   - 水文年1：2020-04 → 2021-03
   - 水文年2：2021-04 → 2022-03
   - ...
4. 计算月平均库容
5. 计算月纳污能力：W = 31.536 × K × V × Cs × b
6. 汇总年合计（12个月相加）

十、代码简化
-----------
水库比河道简单很多：
- 无支流、无分段
- 无流速计算
- 无浓度混合
- 公式直接乘法

估计代码量：hd_的1/3
